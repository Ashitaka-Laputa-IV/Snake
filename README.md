# Snake Game

一个跨平台的贪吃蛇游戏，使用Python和Pygame开发。

## 安装依赖

### Windows
```bash
pip install -r requirements.txt
```

### Linux
```bash
pip install -r requirements.txt
# 或者使用系统包管理器
# Ubuntu/Debian:
sudo apt-get install python3-pygame
# Fedora:
sudo dnf install python3-pygame
```

### macOS
```bash
pip install -r requirements.txt
# 或者使用Homebrew
brew install pygame
```

## 运行游戏

```bash
python snake.py
```

## 游戏控制

- 方向键或WASD：控制蛇的移动方向
- ESC键：退出游戏

## 游戏规则

- 控制蛇吃食物来增长
- 避免撞到墙壁或自己的身体
- 每吃一个食物得1分
